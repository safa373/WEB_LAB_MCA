JavaScript String and Math Methods

String Methods

1. String.prototype.toUpperCase()

Converts a string to uppercase letters.

const str = "hello world";
const upperStr = str.toUpperCase();
console.log(upperStr); // "HELLO WORLD"

2. String.prototype.toLowerCase()

Converts a string to lowercase letters.

const str = "HELLO WORLD";
const lowerStr = str.toLowerCase();
console.log(lowerStr); // "hello world"

3. String.prototype.includes()

Checks if a string contains a specific substring.

const str = "The quick brown fox";
const hasFox = str.includes("fox");
console.log(hasFox); // true

4. String.prototype.indexOf()

Returns the index of the first occurrence of a specified value in a string.

const str = "Hello, world!";
const index = str.indexOf("world");
console.log(index); // 7

5. String.prototype.split()

Splits a string into an array of substrings based on a delimiter.

const str = "apple,orange,banana";
const fruits = str.split(",");
console.log(fruits); // ["apple", "orange", "banana"]

Math Methods

1. Math.random()

Generates a random floating-point number between 0 (inclusive) and 1 (exclusive).

const randomNum = Math.random();
console.log(randomNum); // Output: Random number between 0 and 1

2. Math.floor()

Rounds a number down to the nearest integer.

const num = 4.7;
const floored = Math.floor(num);
console.log(floored); // 4

3. Math.ceil()

Rounds a number up to the nearest integer.

const num = 4.3;
const ceiled = Math.ceil(num);
console.log(ceiled); // 5

4. Math.max()

Returns the largest of the numbers provided.

const maxNum = Math.max(5, 12, 8, 23, 7);
console.log(maxNum); // 23

5. Math.min()

Returns the smallest of the numbers provided.

const minNum = Math.min(5, 12, 8, 23, 7);
console.log(minNum); // 5
